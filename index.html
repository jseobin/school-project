Cloudflare Native Pages + Functions + D1

One stack. One network. One deploy flow.

This project now uses Cloudflare services end to end. Static assets are served by Pages, API routes run on Pages Functions, and application data lives in D1.

Frontend on Pages

Static files deploy from GitHub and fan out to the edge automatically.

Backend on Functions

Requests to /api/* are handled by Cloudflare runtime code in this repo.

Database on D1

Guestbook entries are stored in a Cloudflare SQLite database with zero extra hosting.
