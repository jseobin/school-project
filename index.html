Direct Calculator

26수능 점수만 넣고 바로 합격예측

엑셀 업로드 없이 직접 점수를 입력하면, 26수능 분석기의 환산식과 지원선을 웹에서 바로 계산합니다. 현재 직접 입력 계산은 26수능 데이터셋 기준으로 동작합니다.

데이터 불러오는 중 엑셀 기준 시점 로드 대기
실시간 계산

입력값이 바뀌면 업로드 없이 결과표를 즉시 다시 계산합니다.

원본 지원선 사용

적정, 예상, 소신 기준점은 원본 분석기 결과 시트 기준입니다.

내신 직접 반영

대학별 내신점수 기본값을 유지하거나 필요한 항목만 수정할 수 있습니다.

사용 방법

1. 국어, 수학, 탐구 과목과 점수를 입력합니다.

2. 영어, 한국사, 제2외국어 등급을 입력합니다.

3. 필요하면 내신점수를 조정합니다.

4. 이과 / 문과 결과표에서 환산점수와 지원 가능성을 바로 확인합니다.
