수능 합격예측 분석기 웹

25수능·26수능 고속성장분석기 엑셀 파일을 업로드하면 결과 시트를 웹에서 더 빠르게 검색, 필터, 정렬, 비교할 수 있습니다.

1. 엑셀에서 점수 입력

원본 분석기 파일에 점수를 입력하고 저장합니다.

2. 웹에 업로드

`.xlsx` 또는 `.xlsb` 파일을 그대로 올리면 됩니다.

3. 결과 탐색

이과·문과 탭, 모집군, 지역, 합격가능성 기준으로 즉시 좁힙니다.

지원 파일

`25수능 합격예측 분석기(.xlsx)`와 `26수능 합격예측 분석기(.xlsb)`를 자동 감지합니다.

연도 비교

두 파일을 함께 올리면 동일 모집단위의 지원선 변화를 비교합니다.

개인 결과 유지

웹은 업로드된 파일 안의 계산 결과를 읽기만 하며 원본 파일을 수정하지 않습니다.

분석기 업로드

엑셀에서 계산이 끝난 분석기 파일을 올리면 결과표를 웹 인터페이스로 변환합니다.

파일을 불러오지 않았습니다.

업로드 요약

현재 선택된 연도의 입력 점수와 결과 개수를 보여줍니다.

연도 미선택

수능 입력값

내신 입력값

25 vs 26 비교

같은 모집단위 기준으로 지원선 차이를 비교합니다.

비교 준비 전

결과 필터

연도, 계열, 합격가능성, 모집군, 지역, 검색어로 결과를 좁힙니다.

결과 테이블

행을 클릭하면 오른쪽에서 모집단위 상세와 반영 규칙을 확인할 수 있습니다.
